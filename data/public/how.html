E-Mail als Anhang weiterleiten

Bzw. als .eml-Datei herunterladen

Hier finden sich kurze Anleitungen für die gängigsten Mail-Clients

Gmail (Mobile)

Gmail (Mobile) bietet keine direkte Funktion für „Weiterleiten als Anhang“. Stattdessen:

  • Öffnen Sie die E-Mail, die Sie senden möchten.
  • Tippen Sie auf Weiterleiten.
  • Ändern Sie weder den Betreff noch den Inhalt.
  • Senden Sie die E-Mail direkt an start@mailcheck.help.
Outlook (Mobile)

In der mobilen Outlook-App ist das Weiterleiten als Anhang nicht verfügbar. Bitte:

  • Öffnen Sie die gewünschte E-Mail.
  • Tippen Sie auf Weiterleiten.
  • Lassen Sie Betreff und Inhalt unverändert.
  • Senden Sie die E-Mail an start@mailcheck.help.
Apple Mail (iOS)

Auf iPhone und iPad erlaubt Apple Mail kein direktes Weiterleiten als Anhang. Vorgehensweise:

  • Öffnen Sie die betreffende E-Mail.
  • Tippen Sie auf Weiterleiten.
  • Bearbeiten Sie weder Betreff noch Inhalt.
  • Versenden Sie die E-Mail unverändert an start@mailcheck.help.
Samsung Mail

In der Samsung-Mail-App ist das Weiterleiten als Anhang leider nicht implementiert. Verwenden Sie stattdessen:

  • Öffnen Sie die E-Mail.
  • Tippen Sie auf Weiterleiten.
  • Ändern Sie weder Betreff noch Inhalt.
  • Senden Sie die E-Mail direkt weiter an start@mailcheck.help.
Gmail (Desktop)

In Gmail (Desktop) gibt es kein direktes „Weiterleiten als Anhang“. Um eine E-Mail als separate Datei zu versenden, laden Sie sie zunächst herunter:

  • Öffnen Sie die E-Mail, die Sie anhängen möchten.
  • Klicken Sie oben rechts auf die drei Punkte (Weitere Aktionen) und wählen Sie Nachricht herunterladen. Die E-Mail wird als .eml-Datei gespeichert.
  • Erstellen Sie anschließend eine neue E-Mail.
  • Fügen Sie die .eml-Datei als Anhang hinzu.
  • Geben Sie start@mailcheck.help als Empfänger ein und senden Sie die Nachricht.
  • Ein Betreff ist nicht erforderlich, da wir die E-Mail automatisch analysieren.
Outlook (Desktop)

Outlook (Desktop) bietet eine integrierte Funktion zum Weiterleiten als Anhang:

  • Öffnen Sie Outlook und wählen Sie die gewünschte E-Mail.
  • Klicken Sie auf Weitere Optionen (die drei Punkte) und dann auf Weiterleiten als Anhang.
  • Fügen Sie als Empfänger start@mailcheck.help hinzu und senden Sie die E-Mail.
  • Ein Betreff ist nicht erforderlich, da wir die E-Mail automatisch analysieren.
Thunderbird

Thunderbird kann E-Mails direkt als Anhang weiterleiten:

  • Wählen Sie die gewünschte E-Mail in Thunderbird aus.
  • Klicken Sie in der Menüleiste auf Nachricht > Weiterleiten als Anhang.
  • Fügen Sie start@mailcheck.help als gewünschten Empfänger hinzu und senden Sie die E-Mail.
  • Ein Betreff ist nicht erforderlich, da wir die E-Mail automatisch analysieren.
Apple Mail (Mac)

Apple Mail (Mac) unterstützt das Weiterleiten als Anhang:

  • Wählen Sie die entsprechende E-Mail aus.
  • Klicken Sie oben auf E-Mail als Anhang weiterleiten.
  • Fügen Sie den Empfänger (start@mailcheck.help) hinzu und senden Sie die Nachricht.
  • Ein Betreff ist nicht erforderlich, da wir die E-Mail automatisch analysieren.
Yahoo Mail (Browser)

Yahoo Mail bietet keine direkte Weiterleitung als Anhang. Bitte:

  • Öffnen Sie die E-Mail, die Sie weiterleiten möchten.
  • Klicken Sie auf Weiterleiten.
  • Lassen Sie den Betreff und den Inhalt unverändert.
  • Senden Sie die Nachricht direkt an start@mailcheck.help.
Windows 10/11 Mail (Desktop)

Die Windows Mail-App unterstützt kein Weiterleiten als Anhang. Stattdessen:

  • Öffnen Sie die E-Mail.
  • Klicken Sie auf Weiterleiten.
  • Bearbeiten Sie Betreff und Inhalt nicht.
  • Senden Sie die E-Mail an den gewünschten Empfänger (start@mailcheck.help).
GMX Mail (Browser)

GMX bietet die Möglichkeit, E-Mails als .eml-Datei herunterzuladen. Bitte:

  • Öffnen Sie die E-Mail, die Sie versenden möchten.
  • Klicken Sie auf Drucken/Speichern (siehe Anleitung).
  • Speichern Sie die E-Mail als .eml-Datei.
  • Erstellen Sie eine neue E-Mail und fügen Sie die .eml-Datei als Anhang hinzu.
  • Senden Sie die Nachricht an start@mailcheck.help.
WEB.DE Mail (Browser)

WEB.DE Mail kann E-Mails nicht direkt als Anhang versenden. Alternative:

  • Öffnen Sie die entsprechende E-Mail.
  • Klicken Sie auf Weiterleiten.
  • Bearbeiten Sie nichts an Betreff oder Inhalt.
  • Senden Sie die E-Mail unverändert an start@mailcheck.help.
ProtonMail (Browser)

ProtonMail hat keine Standardfunktion für das Weiterleiten als Anhang. Aus Gründen der Ende-zu-Ende-Verschlüsselung ist ein .eml-Anhang nicht immer praktikabel. Verwenden Sie stattdessen:

  • Öffnen Sie die E-Mail.
  • Klicken Sie auf Weiterleiten.
  • Verändern Sie Betreff und Inhalt nicht.
  • Senden Sie die E-Mail unverändert an start@mailcheck.help.
Hinweis: Möchten Sie die E-Mail dennoch als Anhang versenden, müssten Sie den Text zunächst entschlüsseln und extern als .eml speichern. Dies kann die Sicherheit beeinträchtigen.
Tutanota (Browser)

Tutanota unterstützt ebenfalls kein natives Weiterleiten als Anhang. Nutzen Sie den regulären Weiterleiten-Button:

  • Öffnen Sie die E-Mail.
  • Klicken Sie auf Weiterleiten.
  • Bearbeiten Sie Betreff oder Inhalt nicht.
  • Senden Sie die Nachricht unverändert an start@mailcheck.help.
Zoho Mail (Browser)

Zoho Mail bietet nur begrenzt Optionen zum Weiterleiten als Anhang. Standardmäßig empfiehlt sich:

  • Öffnen Sie die E-Mail.
  • Klicken Sie auf Weiterleiten.
  • Verändern Sie weder Betreff noch Inhalt.
  • Senden Sie die E-Mail unverändert an start@mailcheck.help.
AOL Mail (Browser)

AOL Mail stellt keinen direkten „Weiterleiten als Anhang“-Button zur Verfügung. Führen Sie bitte Folgendes aus:

  • Öffnen Sie die E-Mail.
  • Klicken Sie auf Weiterleiten.
  • Lassen Sie Betreff und Inhalt unverändert.
  • Senden Sie die Nachricht direkt an start@mailcheck.help.
Spark (Mobile/Desktop)

Spark bietet kein direktes Weiterleiten als Anhang. Nutzen Sie stattdessen:

  • Öffnen Sie die gewünschte E-Mail.
  • Klicken bzw. tippen Sie auf Weiterleiten.
  • Bearbeiten Sie nichts an Betreff oder Inhalt.
  • Senden Sie die E-Mail unverändert an start@mailcheck.help.
BlueMail (Mobile/Desktop)

In BlueMail ist das Weiterleiten als Anhang nicht von Haus aus verfügbar. Als Alternative:

  • Öffnen Sie die E-Mail.
  • Tippen bzw. klicken Sie auf Weiterleiten.
  • Bearbeiten Sie Betreff oder Inhalt nicht.
  • Senden Sie die E-Mail unverändert an start@mailcheck.help.
Newton Mail (Mobile/Desktop)

Newton Mail besitzt keine integrierte Weiterleitung als Anhang. Verwenden Sie daher die Standard-Weiterleitung:

  • Öffnen Sie die E-Mail.
  • Klicken oder tippen Sie auf Weiterleiten.
  • Bearbeiten Sie weder Betreff noch Inhalt.
  • Senden Sie die E-Mail unverändert an start@mailcheck.help.
Fastmail (Browser)

Fastmail stellt keine direkte Weiterleitungsfunktion als Anhang bereit. Verwenden Sie hierfür die reguläre Weiterleitung:

  • Öffnen Sie die betreffende E-Mail.
  • Klicken Sie auf Weiterleiten.
  • Bearbeiten Sie nichts am Betreff oder Inhalt.
  • Senden Sie die E-Mail an den gewünschten Empfänger (start@mailcheck.help).
E-Mails werden nach dem verarbeiten direkt gelöscht und Daten nur für die KI-Analyse mit Dritten geteilt.
